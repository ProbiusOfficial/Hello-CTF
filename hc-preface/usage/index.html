    еҰӮдҪ•дҪҝз”Ёжң¬д№ҰВ¶

    Info

    дёәдәҶиҺ·еҫ—жңҖдҪізҡ„йҳ…иҜ»дҪ“йӘҢпјҢе»әи®®жӮЁйҖҡиҜ»жң¬иҠӮеҶ…е®№гҖӮ

    йЈҺж јзәҰе®ҡВ¶

    • ж ҮйўҳеҗҺж ҮжіЁ * зҡ„ж–Үз« е…·жңүдёҖе®ҡйҡҫеәҰпјҢйҖҡеёёж¶үеҸҠдёҖдәӣжҠҖжңҜзҡ„еә•еұӮеҺҹзҗҶпјҢеңЁжңӘе®Ңе…ЁзҶҹжӮүиҜҘж–№еҗ‘еӨ§иҮҙеҶ…е®№ж—¶пјҢйҳ…иҜ»иө·жқҘдјҡжңүиҫғеӨ§йҡҫеәҰпјҢе®№жҳ“йҖ жҲҗдёҚеҘҪзҡ„еӯҰд№ дҪ“йӘҢпјҢжүҖд»ҘиҜҘйғЁеҲҶеҶ…е®№еңЁеүҚжңҹеҸҜд»ҘжҡӮж—¶и·іиҝҮпјҢзӯүеҲ°еҜ№иҜҘж–№еҗ‘жңүдәҶдёҖе®ҡдәҶи§ЈеҗҺеҶҚеӣһжқҘйҳ…иҜ»гҖӮ
    • ж–Үз« дёӯзҡ„йҮҚиҰҒеҗҚиҜҚдјҡз”Ё гҖҢ гҖҚ жӢ¬еҸ·ж ҮжіЁпјҢдҫӢеҰӮ гҖҢи·Ёз«ҷи„ҡжң¬ж”»еҮ» XSS(Cross Site Scripting)гҖҚ гҖӮиҜ·еҠЎеҝ…и®°дҪҸиҝҷдәӣеҗҚиҜҚпјҢеҢ…жӢ¬иӢұж–Үзҝ»иҜ‘д»ҘеҸҠзј©еҶҷпјҢдҫҝеҗҺз»ӯйҳ…иҜ»ж–ҮзҢ®ж—¶дҪҝз”ЁгҖӮ
    • еҠ зІ—зҡ„ж–Үеӯ— иЎЁзӨәйҮҚзӮ№еҶ…е®№жҲ–жҖ»з»“жҖ§иҜӯеҸҘпјҢиҝҷзұ»ж–Үеӯ—еҖјеҫ—зү№еҲ«е…іжіЁгҖӮ
    • дё“жңүеҗҚиҜҚе’Ңжңүзү№жҢҮеҗ«д№үзҡ„иҜҚеҸҘдјҡдҪҝз”Ё вҖңеҸҢеј•еҸ·вҖқ ж ҮжіЁпјҢд»ҘйҒҝе…Қжӯ§д№үгҖӮ

    • жӮЁеҸҜиғҪдјҡйҒҮеҲ°д»ҘдёӢеҮ з§ҚжҸҗзӨәжЎҶпјҢе®ғд»¬еҲҶеҲ«иЎЁзӨәдёҚеҗҢзҡ„еҗ«д№үпјҡ

      Note

      иЎЁзӨәз¬”и®°пјҢжҸҗзӨәдёҖдәӣжіЁж„ҸдәӢйЎ№пјҢжҲ–иҖ…дёҖдәӣе°ҸжҠҖе·§гҖӮ

      Abstract

      иЎЁзӨәж‘ҳиҰҒпјҢз”ЁдәҺеј•з”ЁдёҖдәӣеӨ–йғЁж–ҮзҢ®пјҢжҲ–иҖ…еј•з”ЁдёҖдәӣйҮҚиҰҒзҡ„иҜӯеҸҘгҖӮ

      Info

      иЎЁзӨәдҝЎжҒҜпјҢз”ЁдәҺжҸҗзӨәдёҖдәӣйҮҚиҰҒзҡ„дҝЎжҒҜгҖӮ

      Example

      иЎЁзӨәзӨәдҫӢпјҢз”ЁдәҺдёҫдҫӢиҜҙжҳҺгҖӮ

      Tip

      иЎЁзӨәжҸҗзӨәпјҢз”ЁдәҺжҸҗзӨәдёҖдәӣе°ҸжҠҖе·§гҖӮ

      Success

      дёҚдёҖе®ҡиЎЁзӨәжҲҗеҠҹжң¬ж„ҸпјҢиҜ·жіЁж„ҸиҜҘеӣҫж Үзҡ„иҮӘе®ҡд№үж ҮйўҳгҖӮ

      Question

      иЎЁзӨәй—®йўҳпјҢз”ЁдәҺжҸҗеҮәдёҖдәӣй—®йўҳпјҢдҫӣиҜ»иҖ…жҖқиҖғгҖӮ

      Warning

      иЎЁзӨәиӯҰе‘ҠпјҢжҲ–иҖ…дёҖдәӣйңҖиҰҒжіЁж„Ҹзҡ„ең°ж–№пјҢзЁӢеәҰжҜ”FailureдҪҺгҖӮ

      Failure

      з”ЁдәҺжҸҗзӨәдёҖдәӣеҚұйҷ©зҡ„ж“ҚдҪңгҖӮ

      Danger

      з”ЁдәҺжҸҗзӨәдёҖдәӣйқһеёёеҚұйҷ©зҡ„ж“ҚдҪңгҖӮ

      Bug

      з”ЁдәҺжҸҗзӨәдёҖдәӣеҸҜиғҪдјҡеҜјиҮҙй”ҷиҜҜзҡ„ж“ҚдҪңгҖӮ

      Quote

      е…·дҪ“ж„Ҹд№үз”ұж ҮйўҳеҶіе®ҡ

    еҠЁжүӢе®һж“ҚВ¶

    • жң¬д№Ұзҡ„й…ҚеҘ—йўҳзӣ®зҡ„ йҷ„д»¶ жәҗз Ғ е®№еҷЁ еқҮжүҳз®ЎеңЁGitHub д»“еә“пјҢеҜ№дәҺжҜҸдёӘйўҳзӣ®пјҢжҲ‘д»¬еқҮдјҡжҸҗдҫӣиҜҰз»Ҷзҡ„йўҳи§Је’ҢзҹҘиҜҶзӮ№и®Іи§ЈпјҢжҲ‘д»¬йј“еҠұиҜ»иҖ…еҜ№жҜҸдёӘйўҳзӣ®иҝӣиЎҢжүӢеҠЁеӨҚзҺ°пјҢе®һж“ҚиғҪеҠӣеҜ№дәҺжҜҸдёҖдёӘCTFerд»ҘеҸҠе®үе…Ёд»ҺдёҡиҖ…йғҪиҮіе…ійҮҚиҰҒгҖӮ
    • жңӘжқҘпјҢHello CTF е°ҶдјҡдёҺ NSSCTF е№іеҸ°иҒ”еҠЁпјҢжң¬д№ҰжүҖжңүзҡ„йўҳзӣ®еқҮдјҡеңЁ NSSCTF е№іеҸ°дёҠејҖж”ҫпјҢдҫӣиҜ»иҖ…з»ғд№ гҖӮ

    е–„дәҺжҸҗй—®В¶

    жң¬д№Ұзҡ„жҜҸзҜҮж–Үз« дёӢйғҪжңүиҜ„и®әеҢәпјҢжҲ‘д»¬еҫҲйј“еҠұиҜ»иҖ…еңЁиҜ„и®әеҢәжҸҗй—®и®Ёи®әпјҢеӣ дёәиҝҷдёҚд»…еҸҜд»Ҙеё®еҠ©жӮЁи§ЈеҶій—®йўҳпјҢд№ҹеҸҜд»Ҙеё®еҠ©е…¶д»–иҜ»иҖ…и§ЈеҶій—®йўҳпјҢеҗҢж—¶д№ҹеҸҜд»Ҙеё®еҠ©жҲ‘д»¬е®Ңе–„жң¬д№ҰгҖӮ

    Warning

    еңЁжҸҗй—®еүҚпјҢиҜ·зЎ®дҝқжӮЁе·Із»Ҹйҳ…иҜ»дәҶжң¬д№Ұзҡ„зӣёе…іеҶ…е®№пјҢд»ҘеҸҠзӣёе…ізҡ„ж–ҮзҢ®иө„ж–ҷпјҢе№¶дё”е·Із»Ҹе°қиҜ•иҝҮиҮӘе·ұи§ЈеҶій—®йўҳпјӣиҜҰз»ҶеҸҜд»ҘеҸӮиҖғ жҸҗй—®зҡ„жҷәж…§ иҝҷзҜҮж–Үз« пјҢдёәдәҶжӮЁзҡ„й—®йўҳиғҪеӨҹеҫ—еҲ°жӣҙеҘҪзҡ„и§Јзӯ”пјҢжҲ‘д»¬е»әи®®жӮЁеңЁжҸҗй—®еүҚйҳ…иҜ»иҜҘж–Үз« гҖӮ

    еӯҰд№ и·ҜзәҝВ¶

    1. е…Ҙй—ЁгҖӮжҲ‘д»¬е»әи®®жӮЁе…Ҳйҳ…иҜ» еҝ«йҖҹејҖе§Ӣ иҝҷзҜҮж–Үз« пјҢе»әз«ӢеҜ№CTFзҡ„дёҖдёӘеӨ§дҪ“и®ӨзҹҘпјҢдәҶи§ЈCTFеҗ„дёӘж–№еҗ‘зҡ„еҹәжң¬еҶ…е®№пјҢз„¶еҗҺеҶҚж №жҚ®иҮӘе·ұзҡ„е…ҙи¶ЈйҖүжӢ©дёҖдёӘж–№еҗ‘иҝӣиЎҢж·ұе…ҘеӯҰд№ гҖӮ

    2. еҲ·йўҳгҖӮеҲ·йўҳжҳҜдёҖз§ҚеҫҲеҘҪзҡ„е®һи·өж–№жі•пјҢеңЁеҲ·йўҳдёӯ еӯҰд№ зҹҘиҜҶзӮ№пјҢеҲ©з”ЁзҹҘиҜҶзӮ№пјҢе·©еӣәзҹҘиҜҶзӮ№ еҪўжҲҗдёҖдёӘиүҜжҖ§еҫӘзҺҜпјҢйҖҗжёҗеҺ»жһ„е»әзҹҘиҜҶдҪ“зі»гҖӮ

      • гҖҗе№іеҸ°и®ӯз»ғгҖ‘
        еңЁ еҝ«йҖҹејҖе§Ӣ жҲ‘д»¬жңүз»ҷеҮәдёҖдәӣеҲ·йўҳе№іеҸ°е’ҢеҹәзЎҖйўҳеҚ•пјҢеҸҜд»Ҙд»ҺеҹәзЎҖйўҳеҚ•ејҖе§ӢеҲ·йўҳпјҢдёҚдјҡзҡ„йўҳдјҳе…ҲдҪҝз”Ёжҗңзҙўеј•ж“ҺзӯүдҝЎжҒҜж”¶йӣҶзҡ„жё йҒ“еҺ»еҜ»жүҫеҜ№еә”йўҳзӣ®ж¶үеҸҠеҲ°зҡ„зҹҘиҜҶзӮ№пјҢеҸҜиғҪзҡ„иҖғзӮ№пјҢй’ҲеҜ№жҖ§зҡ„еҺ»еҜ»жүҫи§Јйўҳж–№жі•жҲ–иҖ…жҖқиҖғи§Јйўҳж–№жі•пјҢеҰӮжһңжӮЁеңЁе°қиҜ•дәҶдёҠиҝ°ж–№жі•еҗҺд»Қз„¶ж— жі•и§ЈеҶіпјҢеҸҜд»ҘеҜ»жүҫйўҳзӣ®зҡ„WriteUpпјҢеҜ№з…§WPиҝӣиЎҢеӯҰд№ гҖӮ

      • гҖҗжҜ”иөӣи·ҹиҝӣгҖ‘
